YOY Changes Waterfall Chart 


To be able to customize the YOY Changes waterfall chart we to create a table first as shown below with detailed Instructions:

Waterfall Table Settings

Go to insert and click on Enter Data

and create the table as directed in the GIF

# MeasureID: 1,2,3,4

# Category :

- Acc YoY%

- Dths YoY%

- AP Dths YoY%

- TO YoY%

- OV YoY%

 





MEASURE: YOY Changes% =

SWITCH (

    SELECTEDVALUE ( 'WFall_Table'[MeasureID] ),

    1, [Acc YoY%],

    2, [Dths YoY%],

    3, [AP Dths YoY%] ,

    4, [TO YoY%],

    5, [OV YoY%]

)
