Creating Refresh Date Table 13

1. Create a New Table:

   - Go to the `Modeling` tab.

   - Click on `New Table`.

2. Enter DAX Formula:

   - In the formula bar, enter the following DAX expression:

     RefreshDateTable =

     VAR RefreshDate = NOW()

     RETURN

     ADDCOLUMNS(

         CALENDAR(RefreshDate, RefreshDate),

         "Refresh Date", RefreshDate

     )

3. Create a Measure for Refresh Date:

   - Go to the `Modeling` tab.

   - Click on `New Measure`.

   - Enter the following DAX formula in the formula bar:

     Last Refreshed =

     MAX(RefreshDateTable[Refresh Date])

   This measure extracts the latest refresh date from your refresh date table.
