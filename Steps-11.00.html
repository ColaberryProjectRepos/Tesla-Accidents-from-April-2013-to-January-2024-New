Loading Data into PowerBI 11

Detailed Instructions:

1. Click on the link to the raw file dataset and save it to your device.

2. Open Power BI Desktop and go to the Home ribbon, then click on "Get data."

3. From the drop-down list, select "Text/CSV Workbook" and click "Connect."

4. Select the file named " TeslaAccidents_FinalDataset."

5. When the preview appears, click "Load."
