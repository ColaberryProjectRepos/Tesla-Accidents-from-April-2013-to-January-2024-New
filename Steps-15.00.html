Follow the below instructions and the GIF for more details: 


1. Create a New Table for Measures:

   - Go to the `Modeling` tab.

   - Click on `New Table`.

2. Enter a Dummy DAX Formula:

   - In the formula bar, enter a simple DAX formula to create an empty table. For example:

     MeasuresTable = DATATABLE("Name", STRING, {})

   This will create an empty table named `MeasuresTable`.

3. Create New Measures:

   - Once the `MeasuresTable` is created, you can start adding measures to this table.

   - Go to the `Modeling` tab.

   - Click on `New Measure`.

4. Define Your Measure:

   - In the formula bar, enter your DAX formula for the measure. For example:

     %DeathRate = DIVIDE([Dths], [Acc], 0)
