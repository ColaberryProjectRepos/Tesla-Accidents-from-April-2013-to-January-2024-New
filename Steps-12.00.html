Creating a Calendar Table 12

To create a calendar table we use the following:

Calendar =

ADDCOLUMNS(

    CALENDAR(DATE(2013, 4, 1), DATE(2024, 1, 30)),

    "Year", YEAR([Date]),

    "Month Number", MONTH([Date]),

    "Month Name Short", FORMAT([Date], "MMM"),

    "Month Name Long", FORMAT([Date], "MMMM"),

    "Day of Week Number", WEEKDAY([Date]),

    "Day Name Short", FORMAT([Date], "ddd"),

    "Day Name Long", FORMAT([Date], "dddd"),

    "Quarter", QUARTER([Date]),

    "Week of Year", WEEKNUM([Date], 2) // Adjust the week number system as needed

)
